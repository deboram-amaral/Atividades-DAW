3. TAGs de texto

Voltar

Dentro do body (tudo que será exibido ao usuário) é possível inserir títulos, subtítulos, parágrafos, citações, imagens e uma gama de outros elementos. Aqui falaremos das principais tags de texto.

Falando sobre os h's

Utilizamos os h's para títulos e subtítulos. Sâo 6 opções (h1 ao h6) e o que muda entre eles é o tamanho. Logo, você pode colocar o título principal da sua página no h1 e depois subtítulos num tamanho um pouco menor, por exemplo, utilizando os demais níveis. Exemplo: < h1 > Desenvolvimento WEB com HTML < / h1 > / < h2 > Tudo que você precisa saber para desenvolver uma página na internet utilizando HTML! < / h2 >

Parágrafos

Para inserir parágrafos, utilizamos a tag < p >. Exemplo: < p > insira seu parágrafo aqui < / p > .

Citações

Para inserir citações, utilizamos a tag < blockquote >.

Dessa forma, o texto ficará com uma margem maior à esquerda, configurando uma citação. Exemplo: < blockquote > insira sua citação aqui < / blockquote > .

Itálico, negrito e sublinhado

  • Para deixar o texto em itálico, utilizamos a tag < i > . Exemplo: < bloquequote > < i > insira seu subtítulo aqui < / i > < / bloquewuote > .
  • Para deixar o texto em negrito
    • , utilizamos a tag < strong > . Exemplo: < p > < strong > insira seu parágrafo aqui < / strong > < / p > .
  • Para deixar o texto sublinhado
    • , utilizamos a tag < u > . Exemplo: < h2 > < u > insira seu subtítulo aqui < / u > < / h2 >.

4. Listas Ordenadas e Não Ordenadas

Voltar

Ordenadas

Nas listas ordenadas, os ítens são numerados em órdem crescente. Para inserir uma lista ordenada, utilizamos a tags < ol > para iniciar a lista e < li > ao início de cada ítem. Exemplo:
< ol >
< li > Ítem 1 < / li >
< li > Ítem 2 < / li >
< li > Ítem 3 < / li >
< / ol >

Não ordenadas

Nas listas não ordenadas, os ítens recebem indicadores. No caso do html, com uma bolinha (tópico). Para inserir uma lista não ordenada, utilizamos as tags < ul > para iniciar a lista e < li > ao início de cada ítem. Exemplo:
< ul >
< li > Ítem 1 < / li >
< li > Ítem 2 < / li >
< li > Ítem 3 < / li >
< / ul >

5. Links

Voltar

Para inserir um link utilizados a tag < a href="cole aqui o endereço do site"> insira aqui o texto que será exibido para o usuário (encurtador) < / a >. O "a" da tag vem âncora. Ou seja, o link vai ancorar em algum lugar. Utilizando a tag, quando você abrir sua página no navegador e clicar no encurtador do link, será direcionado para o endereço na mesma página. Caso queira que abra em uma página diferente, pode utilizar a tag < target = " blank " > .
